# 개미의 일기장

우리는 경제적 자유라는 달콤한 꿈을 위해 재태크를 합니다. 하지만 재태크는 너무나도 긴 호흡으로 진행되어 재태크가 제대로 하고 있는 것인지 알기 어렵고, 자칫 잘못하면 재태크가 아닌 무지성, 투기성 투자가 되버릴 수도 있습니다. 이에 우리는 재태크를 관리하고자 합니다.

하지만 재태크를 기록하는 것은 꾸준한 관심이 필요하고, 개인의 노력과 의지가 필요합니다.
누군가는 이러한 재태크 활동이 재미없게 느껴질 수도 있습니다.
체계적으로 재태크 기록을 하고 싶지만, 수기로 작성해아 하는 것에 피로감을 느끼기도 합니다.

게으른 개미들을 위한 재미있고 효율적인 재태크 관리 앱을 경험해 보세요.

## 누구를 위한 앱인가

- 재태크를 조금이라도 재미있게 재태크를 하고 싶은 사람
- 재태크를 체계적으로 기록하고 싶지만 수기로 엑셀을 작성하기는 귀찮은 사람
- 자산 관리 현황을 한 눈에 보고 싶은 사람

## 주요 기능
- 재태크 포트폴리오를 만들고 포트폴리오를 기준으로 재태크 현황을 관리해요
- 재태크 내역을 쉽고 간편하게 기록해요
- [preview] 연간 재태크 목표를 정하고 이를 달성해보아요
- [preview] 물타기 실력, 주요 지표 벤치마킹, 재태크 알림이 등 다양한 기능들로 재태크를 재미있게 해보아요
- [preview] 어려운 재무제표를 쉽게 요약하고, 자동으로 가치평가를 계산받아 보아요
